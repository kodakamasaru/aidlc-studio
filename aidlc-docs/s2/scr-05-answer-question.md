# SCR-05: Q 回答

## メタ
- 親: [s2/index.md](./index.md)
- 対応 US: US-12
- 版: v0.0.1
- ステータス: 確定

## 目的
AI が生成中に発した質問に人間が回答し、Run を resume させる画面。

## 主要 UI 要素
- 質問本文 + 文脈リンク(該当 Cycle / ステップ / 関連成果物への導線)
- 回答入力(複数行テキスト)
- 送信ボタン

## 状態 (data-state)
- default: 質問本文と回答入力欄を表示。送信ボタンは回答入力で活性。

## 遷移
- IN: SCR-03(Q 待ちカードを開く)
- OUT: 送信 → SCR-02(回答を渡して Run 再開)

## 備考(挙動 / native / a11y)
- 回答は複数行・コードブロック可。
- 文脈リンクから質問の背景(該当ステップ出力)を確認できること。
- 送信後は当該 Q 待ちカードが Inbox から消える。
