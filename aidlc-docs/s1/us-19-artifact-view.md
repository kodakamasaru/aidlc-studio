## 3 観点

### なぜするか (Why)
Artifact(US/Mock/Flow/UoW/…)は各ステップの成果物であり、外部記憶として後工程 Agent や人間レビューの一次資料になる。サイト上で閲覧できないと、人間は IDE を開く羽目になり「コードを見ない」前提が崩れる。

### UX へのインパクト
ユーザーは Cycle 内の各ステップ成果物を一覧から開き、md/Mermaid/screenshot を整形済みで読める。過去工程の意図をいつでも参照しながら判断できる。

### 受け入れ条件 (AC)
- [v0.0.x] Cycle に紐づく各ステップの Artifact を一覧表示できる
- 各 Artifact の中身(md/Mermaid/screenshot)を整形して閲覧できる
- ステップ種別(S1〜S7)で Artifact を絞り込める
- 最新版と過去版(手戻りで再生成された版)を区別して参照できる
