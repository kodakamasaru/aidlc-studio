## 3 観点

### なぜするか (Why)
Cycle は AI-DLC の実行単位であり、Task は単独では開始しない(R-01)。1 つ以上の Task を束ねた Cycle を作ることが、AI-DLC を回す前提になる。MVP の入口。

### UX へのインパクト
ユーザーは Task を選んで Cycle を作成でき、最小では単一 Task でも開始できる。これで「これから回す塊」が明確な実体として立ち上がる。

### 受け入れ条件 (AC)
- [v0.0.1] 1 つ以上の Task を含む Cycle を作成できる(最小は単一 Task)
- 作成された Cycle に一意 ID・名前・作成日時・初期ステータス(未開始)が付与される
- Task を 1 件も含まない Cycle は作成できない
- 作成直後の Cycle は「開始可能」状態として表示される
