# SCR-03: Human Inbox

## メタ
- 親: [s2/index.md](./index.md)
- 対応 US: US-12, US-13
- 版: v0.0.1
- ステータス: 確定

## 目的
人間の判断を待つカードを 1 箇所に集約して捌くハブ。Q 待ちは回答へ、レビュー待ちは詳細へ振り分ける。

## 主要 UI 要素
- 待ちカード一覧(各カード: 種別バッジ = Q待ち / レビュー待ち、Cycle 名、ステップ、生成日時)
- カードを開く導線(種別で遷移先が分岐)

## 状態 (data-state)
- empty: 待ちカードなし。「いま捌くものはありません」を表示。
- list: Q待ち / レビュー待ちカードが混在で並ぶ。種別バッジで識別可能。

## 遷移
- IN: SCR-02(Q 発生 / ステップ完了でカード生成)/ グローバルナビ
- OUT: Q待ちカード → SCR-05(Q 回答)
- OUT: レビュー待ちカード → SCR-04(レビュー詳細)

## 備考(挙動 / native / a11y)
- カードは種別で視覚的に区別(色 / アイコンは S2.5)。
- 並びは生成日時など一定順序。捌いたカードは一覧から消える(or 既処理表示)。
- Inbox が製品のハブ。ここを起点に Q 回答・レビューの両ループが回る。
