## 3 観点

### なぜするか (Why)
走行中の Cycle を止めたい(別件優先・暴走の疑い)場面や、進行を見守りたい場面がある。一時停止と注視ができないと、人間は走り出した AI を制御できず、リソースや方向性のリスクを管理できない。

### UX へのインパクト
ユーザーは走行中 Cycle を一時停止して後で再開でき、また注視モードで進行ステップやログをリアルタイムに眺められる。AI に任せきりにせず、必要なときだけ手綱を握れる。

### 受け入れ条件 (AC)
- [v0.0.x] 実行中の Cycle を一時停止でき、停止状態が表示される
- 一時停止した Cycle を再開できる
- Cycle を注視モードにすると現在ステップ/進行ログをリアルタイムに観察できる
- 一時停止/再開の操作が履歴に残る
