## 画面遷移フロー

```mermaid
flowchart LR
  CYCLES["サイクル一覧"]
  DETAIL["サイクル詳細<br/>(進行可視化)"]
  INBOX["受信箱"]
  REVIEW["レビュー詳細<br/>★スクショ実画像"]
  QA["Q回答"]
  STEPS["ステップ構成ビュー<br/>★作成時スナップショット"]

  CYCLES --> DETAIL
  CYCLES --> STEPS
  DETAIL --> INBOX
  INBOX -->|レビュー依頼| REVIEW
  INBOX -->|質問| QA
  REVIEW -->|承認/却下| DETAIL
  QA -->|回答| DETAIL
```

> フロー自体は v0.0.2 から不変。**live 経路でも人間の導線(Inbox を捌く)は同じ**であることが S2 の確認点。
