## 目的
サイクル作成時に default を DB へ snapshot コピーする(US-02)。この挙動を人間に誤解させないため、構成ビューに「**このサイクルのステップは作成時点で確定。以後デフォルトを変えても、このサイクルには波及しない**」を一文で明示する。新規画面は作らない。

## 主要要素
- **サイクルのステップ構成ビュー**(既存): 上部に注記バナー(作成時スナップショット)。中身(S1〜S12 の構成)は既存描画のまま。表示元が DB snapshot に切り替わる(画面構造は不変)。
- **設定 > デフォルトのステップ構成**(既存): 「ここでの編集は **これから作る**サイクルに反映。既存サイクルは作成時点に固定」を注記。
