## 視覚契約(差分)
- バッジは v0.0.4 デザインシステムの既存トークンを使う。**新規色・新規ラベルは追加しない**。
- 状態マッピング: run=running の間は `pill running`(実行中 / teal #2dd4bf)。**レビュー emit 後は `pill review`(できあがりの確認 / purple #c084fc)** へ切替。
- ラベルは既存 badge「◎ できあがりの確認」と本文 CTA「できあがりを確認する」に揃える。

## native 固有挙動
- 対象は web(ローカル常駐)。native 固有挙動なし。

## a11y
- バッジは色だけで状態を区別しない: ラベル文言(「実行中」/「できあがりの確認」)を必ず併記。
- スクリーンリーダ向けに状態をテキストで読み上げ(aria-label = ラベルと同義)。
- 色コントラスト: review(#c084fc)on review-soft 背景で WCAG AA を満たす(既存 v0.0.4 トークンで担保済)。

## gesture
- tap: CTA「できあがりを確認する」でレビュー詳細へ遷移。
- バッジ自体は非インタラクティブ(状態表示のみ)。

## motion
- レビュー emit でバッジが切り替わる瞬間: 150ms ease-out で色フェード(teal→purple)。急な点滅を避ける。
- **HTML の transition は参考にされない**。motion 意図はこの文字記述が契約。
