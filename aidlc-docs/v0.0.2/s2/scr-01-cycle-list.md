# SCR-01: サイクル一覧(更新) — v0.0.2

## メタ
- 親: [s2/index.md](./index.md)
- 対応 US: [US-06](../s1/us-06-step-custom-ui.md)
- ステータス: 確定

## 目的

v0.0.1 のサイクル一覧画面に **Step定義エディタ** を追加する。ユーザーはサイクル開始前に、各 Step の契約(Output / Verification / HumanGate / Escalation)と execMode、成果物 Profile を画面から編集できる。**サイクル開始後は読み取り専用**になる。

## 主要要素

### A. サイクルカードへの Step定義リンク (US-06)

各サイクルカードに「Step定義」を開くアクションを追加。

- **未開始サイクル**: 「Step定義を編集」(編集可能)
- **開始済みサイクル**: 「Step定義を表示」(読み取り専用)
- Step は可変(S1〜S7 とは限らない)。サイクルごとに任意の Step 構成を持つ

### B. Step定義エディタ (US-06)

サイクルカードの「Step定義」リンクから **専用ビュー(フルページ / 例: /cycles/:id/steps)に画面遷移**する。Step ごとに独立した契約を編集。戻るボタンでサイクル一覧に復帰。

- **Step リスト**: そのサイクルが持つ Step の一覧(追加・削除・並び替え可能)
- **Step ごとの契約**: 4契約タイプ + execMode + 成果物 Profile
- **永続化**: API 経由で保存。次回 Run 実行時に反映
- **編集ロック**: サイクル開始後は全フォームが disabled

## この画面固有の 質疑応答ログ

### Q-01 — Step定義エディタはインライン展開かモーダルか専用画面か？
- **回答**(ユーザー記入):
  > 画面遷移してもいいのでは。
- **確定**(AI 記入):
  > サイクルカードの「Step定義」リンクから **専用ビュー(フルページ)に画面遷移**する。Step が多くてもフル幅で編集でき、サイクルとの紐付きも URL(例: /cycles/:id/steps)で明確。SCR-01 の付随ビューとして扱い、新規 SCR 番号は振らない。戻るボタンでサイクル一覧に復帰。

## この画面固有の AI が独自に決めたこと と 理由

### D-01 — Step定義エディタを SCR-01(サイクル一覧)に配置する
- **理由**: Step定義はサイクル開始前に決めるもの。開始後は不変。サイクルカードに「編集/表示」リンクを置き、未開始は編集可・開始済は読み取り専用に出し分ける。これによりサイクル詳細画面(SCR-02)は実行状況の表示に専念できる。
- **判断**(ユーザー記入): 承認 | 上書き | 保留
- **上書き内容**(上書き時のみ):

### D-02 — Step は可変(追加・削除・並び替え可能)にする
- **理由**: S1 で「Step は可変、S1〜S7 とは限らない」が確定。Step リストを編集可能にし、各 Step が独立した契約を持つ。並び替えはドラッグ&ドロップ。
- **判断**(ユーザー記入): 承認 | 上書き | 保留
- **上書き内容**(上書き時のみ):

### D-03 — Step ごとに独立した契約を持つ(サイクル共通ではない)
- **理由**: S2 Q-01 で「ステップごとに独立」が確定。各 Step が固有の Output/Verification/HumanGate/Escalation 契約と execMode、Profile を持つ。エディタは Step を選択して個別編集する形。
- **判断**(ユーザー記入): 承認 | 上書き | 保留

## この画面固有の 棄却した案

### R-01 — Step定義をインライン展開 / モーダルで開く
- **棄却理由**: S2 Q-01(本画面)で「専用ビューに画面遷移」に決定。Step が多いとインラインは長くなり、モーダルは幅が足りない。フルページ遷移なら Step リスト + 契約編集を余裕を持って配置できる。なお独立 SCR 番号は振らず、SCR-01 配下のビュー(/cycles/:id/steps)とする。

### R-02 — サイクル開始後も Step定義を編集可能にする
- **棄却理由**: 実行中に契約を変えると、すでに実行した Step と矛盾する。S2 Q-01 で「開始後は編集不可」が確定。読み取り専用で履歴として表示する。
