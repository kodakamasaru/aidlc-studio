Motion (意図 — 文字で / 値は参考)

  • hover / focus: 150ms ease-out(背景・境界・shadow)
  • パネル / モーダル出現: 200ms cubic-bezier(0.16,1,0.3,1) で fade + 4px slide-up
  • GEN→EVAL role 切替: バッジ色を 200ms ease-out でクロスフェード(GEN indigo → EVAL violet)
  • deterministic gate progress: バーを 線形に伸長 / 各 block check は 120ms で done 色へ
  • running 状態: dot の pulse 1.6s ease-out infinite / spinner 0.8s linear
  • resume 待ち: skeleton shimmer 1.4s ease-in-out infinite
  • ステップ完了でパイプライン node が done 色へ 250ms ease-out で遷移
