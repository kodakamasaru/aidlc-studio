│ │ 食費   2026-06-18   ¥1,200      │ │

│ ├─────────────────────────────────┤ │

│ │ 映画チケット                    │ │

**状態一覧:**

**インタラクション:**
- 行を長押し → 「削除」「詳細」メニューが表示される(US-02)
- 「＋ 支出を記録する」ボタン → SCR-02 に遷移
