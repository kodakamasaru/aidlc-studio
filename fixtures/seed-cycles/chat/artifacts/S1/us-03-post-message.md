# US-03: チャンネルにメッセージを投稿する

## 3 視点

### なぜするか (Why)
チャットの本質的な価値はメッセージを送って他のメンバーに読んでもらうことにある。投稿ができなければチャンネルは空の器にすぎない。

### UX へのインパクト
テキストを入力して Enter(または送信ボタン)を押すと即座にタイムラインの末尾に自分のメッセージが表示される。他のメンバーのメッセージは上からの時系列で並ぶ。チャンネルを開いている間は新着メッセージがリアルタイムで末尾に追加される。

### 受け入れ条件 (AC)
- メッセージ入力欄にテキストを入力し Enter または「送信」ボタンで投稿できる
- Shift+Enter で改行できる
- 空文字・空白のみのメッセージは投稿できない(送信ボタンが非活性)
- メッセージは最大 4000 文字
- 投稿成功後、入力欄がクリアされタイムラインに自分のメッセージが表示される
- タイムラインには投稿者名・アイコン・投稿日時・本文が表示される
- 他メンバーからの新着メッセージはリアルタイムで反映される(WebSocket)
- 自分が投稿したメッセージには「削除」アクションが表示でき、削除後はタイムラインから消える
