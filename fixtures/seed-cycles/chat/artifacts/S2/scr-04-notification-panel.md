# SCR-04: 通知一覧パネル

## メタ
- 親: 画面要素の一覧
- 対応 US: [US-05](../S1/us-05-mention-notification.md)
- ステータス: 確定

## 目的
自分宛ての@メンション通知を一覧し、クリックで元メッセージへジャンプする。

## 主要要素
- 表示要素: 通知カード(投稿者・チャンネル名・本文スニペット・日時・既読/未読状態)
- アクション: 通知クリック(対象チャンネルへ遷移)、パネル外クリックで閉じる

## モック (ASCII)

```
                        +----------------------------+
                        | 通知                        |
                        +----------------------------+
                        | 🔵 田中 太郎 が #general で |
                        |   あなたをメンションしました |
                        |   「@山田 デザインのレビュー |
                        |    お願いできますか？」      |
                        |   10:28 · 今日              |
                        +----------------------------+
                        | ✓ 佐藤 次郎 が #dev-team で |
                        |   あなたをメンションしました |
                        |   「@山田 PR のレビュー済み  |
                        |    ですか？」                |
                        |   昨日 16:43                |
                        +----------------------------+
```
