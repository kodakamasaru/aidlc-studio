# SCR-01: メインレイアウト(サイドバー + タイムライン)

## メタ
- 親: 画面要素の一覧
- 対応 US: [US-02](../S1/us-02-join-channel.md), [US-03](../S1/us-03-post-message.md), [US-04](../S1/us-04-unread-count.md)
- ステータス: 確定

## 目的
アプリの常時表示ベース画面。左サイドバーにチャンネル一覧と未読バッジ、右コンテンツエリアに選択中チャンネルのメッセージタイムラインを表示する。

## 主要要素
- 入力フィールド: メッセージ入力エリア(複数行対応)
- アクション: 送信ボタン、チャンネル作成ボタン、チャンネル追加ボタン、ベルアイコン、検索アイコン
- 表示要素: チャンネル一覧(名前 + 未読バッジ)、メッセージリスト(投稿者 + 日時 + 本文)、ヘッダー(チャンネル名)

## モック (ASCII)

```
+------------------+-----------------------------------------------+
| 🏠 社内チャット  |  # general                         🔔 🔍      |
+------------------+-----------------------------------------------+
| チャンネル       |  田中 太郎 10:23                               |
| # general     3 |  おはようございます！本日のスタンドアップは     |
| # random      1 |  10:30 からです。よろしくお願いします。          |
| # dev-team      |                                                 |
| # design        |  山田 花子 10:25                               |
|                  |  了解です！参加します。                          |
| [+ チャンネル作成]|                                                 |
| [チャンネルを追加]|  田中 太郎 10:28 @山田                          |
|                  |  山田さん、デザインのレビューお願いできますか？   |
|                  |                                                 |
|                  | ─────────────────────────────────────────────  |
|                  | [メッセージを入力...                    [送信]]  |
+------------------+-----------------------------------------------+
```
