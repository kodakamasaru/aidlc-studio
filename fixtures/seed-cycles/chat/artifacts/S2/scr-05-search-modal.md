# SCR-05: メッセージ検索モーダル

## メタ
- 親: 画面要素の一覧
- 対応 US: [US-06](../S1/us-06-message-search.md)
- ステータス: 確定

## 目的
キーワードでメッセージを全文検索し、結果から元の発言場所へジャンプする。

## 主要要素
- 入力フィールド: 検索キーワード入力(Enter で実行)
- 表示要素: 検索結果カード(チャンネル名・投稿者・日時・本文スニペット)、件数表示、0件メッセージ
- アクション: 結果クリック(対象チャンネルへ遷移)、×閉じるボタン

## モック (ASCII)

```
+--------------------------------------------------+
|  🔍 [スタンドアップ                      ] Enter|
|  3 件のメッセージ                             ×  |
+--------------------------------------------------+
|  # general  田中 太郎  2026-05-12 10:23          |
|  おはようございます！本日の**スタンドアップ**は  |
|  10:30 からです。よろしくお願いします。          |
+--------------------------------------------------+
|  # general  山田 花子  2026-05-11 09:55          |
|  **スタンドアップ**の時間を 10:00 に変更でき    |
|  ますか？                                        |
+--------------------------------------------------+
|  # dev-team 鈴木 一郎  2026-05-10 18:01          |
|  明日の**スタンドアップ**欠席します。議事録      |
|  を共有してください。                            |
+--------------------------------------------------+
```
