# SCR-03: チャンネルブラウザモーダル

## メタ
- 親: 画面要素の一覧
- 対応 US: [US-02](../S1/us-02-join-channel.md)
- ステータス: 確定

## 目的
参加可能な全チャンネルを一覧し、未参加チャンネルに参加する。

## 主要要素
- 表示要素: チャンネルカード(名前・説明・メンバー数・参加状態)
- アクション: 「参加する」ボタン(未参加のみ活性)、×閉じるボタン

## モック (ASCII)

```
+----------------------------------------------+
|  チャンネルを追加する                     ×  |
+----------------------------------------------+
|  ┌──────────────────────────────────────┐    |
|  │ # general                            │    |
|  │ 全体連絡・日報・スタンドアップ用      │    |
|  │ 12 人のメンバー           [参加中]   │    |
|  ├──────────────────────────────────────┤    |
|  │ # dev-backend                        │    |
|  │ バックエンド開発の議論               │    |
|  │ 5 人のメンバー            [参加する] │    |
|  ├──────────────────────────────────────┤    |
|  │ # design                             │    |
|  │ UI/UX デザインレビュー               │    |
|  │ 4 人のメンバー            [参加中]   │    |
|  └──────────────────────────────────────┘    |
+----------------------------------------------+
```
