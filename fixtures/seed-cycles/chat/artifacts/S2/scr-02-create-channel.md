# SCR-02: チャンネル作成モーダル

## メタ
- 親: 画面要素の一覧
- 対応 US: [US-01](../S1/us-01-create-channel.md)
- ステータス: 確定

## 目的
新しいチャンネルの名前と説明を入力して作成する。

## 主要要素
- 入力フィールド: チャンネル名(必須)、説明(任意)
- アクション: 「作成する」ボタン、キャンセルボタン、×閉じるボタン
- 表示要素: バリデーションエラーメッセージ

## モック (ASCII)

```
+--------------------------------------+
|  チャンネルを作成する             ×  |
+--------------------------------------+
|  チャンネル名 *                       |
|  [例: project-alpha              ]   |
|  ※ 英数字・日本語・- _ のみ使用可    |
|                                       |
|  説明(任意)                           |
|  [このチャンネルの用途を入力...   ]   |
|                                       |
|  [キャンセル]          [作成する]     |
+--------------------------------------+
```
